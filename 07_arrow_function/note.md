# 07 arrow function

"Bentuk lain yang lebih ringkas dari Function Expression." - MDN

```javascript
// Function Expression
let tampilPesan = function(nama) {
    console.log(`Helo, ${nama}!`);
}

tampilPesan('Rizqi');


// Arrow Function
let tampilPesan = (nama) => {
    console.log(`Helo, ${nama}!`);
}

tampilPesan('Rizqi');
```
